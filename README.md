# 🌌 AURA - Modern Social Experience

**AURA** ek premium social media platform hai jise modern web technologies aur "Glassmorphism" design language ke saath banaya gaya hai. Ye users ko ek immersive aur visually stunning environment provide karta hai jahan wo Stories, Reels, aur Posts ke zariye connect kar sakte hain.

---

## ✨ Key Features

- 📱 **Glassmorphism UI:** Tailwind CSS aur Backdrop filters ka use karke ek translucent, premium look.
- 📖 **Stories:** Vertical scrolling stories with interactive like and comment features.
- 🎥 **Reels:** Short-form video content support.
- 📰 **Dynamic Feed:** Real-time posts with image/video media integration.
- ⚙️ **Profile Management:** Fully customizable user profiles with bio and avatar updates.
- 🛡️ **Secure Auth:** Bcrypt v5+ based password hashing aur secure session management.
- 🎨 **Responsive Design:** Har device (Mobile, Tablet, Desktop) par seamless experience.

---

## 🚀 Tech Stack

**Frontend:**
- EJS (Embedded JavaScript Templates)
- Tailwind CSS (Styling)
- Plus Jakarta Sans (Typography)

**Backend:**
- Node.js & Express.js
- MongoDB & Mongoose
- Bcrypt (Security)
- Dotenv (Environment Management)
